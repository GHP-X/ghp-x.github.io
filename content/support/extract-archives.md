# **✦ How To Extract**

## **Windows**

Install [**7-Zip**](https://www.7-zip.org) or alternative software. For windows users, we recommend [**7-Zip**](https://www.7-zip.org) as it is both free and open source. Alternatives include NanaZip and WinRAR.

#### [**Click here**](https://files.catbox.moe/v7umjd.mp4) for a video tutorial.

1.  Right click the file you wish to open
2.  Select 7-ZIP — under “show more options” on windows 11.
3.  Extract here > Enter password > OK **(password: `star`)**
4.  Proceed with your new, unzipped folder.

## **MacOS**

Install [**Keka**](http://d.keka.io/) or alternative software. For Mac users, we recommend [**Keka**](http://d.keka.io/) as it is both free and open source. Alternatives include The Unarchiver and PeaZip.

> Note: Only the AppStore version is paid, while the [download](https://ibb.co/Q3fX1cVy) on their site is free.

#### [**Click here**](https://files.catbox.moe/5t0n58.mp4) for a video tutorial.

1.  Right click the file you wish to open
2.  Select open with > choose "Keka"
3.  Select location if asked > Enter password > Done **(password: `star`)**
4.  Proceed with your new, unzipped folder.

Archive utility can open encryptyed files in .zip format, but not .7z. Files will vary.
