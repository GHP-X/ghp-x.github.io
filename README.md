# ghp.wtf

Premium editing resources for After Effects. Curated software, plugins, guides, and solutions for creators at every level.

## What's Inside

- **Guides** — Step-by-step tutorials covering installation, setup, troubleshooting, and best practices
- **Resources** — Curated websites, tools, and references for After Effects work
- **Software** — Downloads and information organized by platform (Windows, macOS)
- **Extensions** — Scripts and CEP extensions to extend After Effects capabilities
- **Support** — Solutions for common issues and problems
- **Archive** — Historical content and legacy information

## Tech Stack

- **Framework:** Next.js 16+ (App Router) with React 19
- **Styling:** Tailwind CSS v4
- **Content:** Markdown-based with remark/turndown pipeline
- **Typography:** Prose plugin for readable markdown rendering
- **Search:** Built-in command palette with full-text indexing
- **Theme:** Dark/light mode support via next-themes

### Content

Add markdown files to the `content/` directory structure. The build pipeline automatically processes and indexes content for search.

```
content/
├── guides/
├── resources/
├── support/
├── categories/
└── ...
```

## Design System

Clean, minimal interface with:
- Professional color palette (B799FF primary accent)
- Subtle shadows and depth
- Responsive grid layouts
- Accessible dark/light modes
- Smooth interactions without excess animation
